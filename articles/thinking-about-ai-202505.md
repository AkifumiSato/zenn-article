- 序文
  - 日々AIが自分の仕事や生活に入ってくる割合が増えてきた
  - 開発業務はもちろん、キャリアや子供の相談といったものまで幅広い
  - Googleで検索することがとても減った
  - AIの進化は目覚ましく、状況が四半期くらいで大きく変わる
  - 今時点でAIに対して自分が考えていることが増えてきたので、整理するために記事にします
  - 参考になるかはわからない、本当に考えていることや感じていることを吐き出すだけ
- 使ってるツール
  - Gemini
  - Github Copilot
  - v0（ご無沙汰）
  - Chat GPT
- AIに対する僕の認識
  - 得意なこと
    - 論理的な思考能力は非常に高い
    - Webで入手できる情報に強い
    - 穴埋め形式の問いにとても強い
  - 苦手なこと
    - 問いを引き出す丁寧さ
    - 時間と共に変化する情報への適用
    - 人間にAIを活用させること
  - 人間側が苦手なこと
    - 課題の構造化
    - 価値観や期待値の言語化
    - 回答に対する評価
- 個人的な使い所とコツ
  - 日々の疑問解消
    - 些細な疑問
    - 子供の食生活や病気
    - ストレスの言語化
  - エンジニアリング
    - 設定ファイルの相談
    - 参考実装がない時: Copilot（Ask）による補完
    - 参考実装がある時: Copilot（Agent）による実装
    - ライブラリ調査や仕様調査
  - キャリアや人生設計の壁打ち
    - 問いや言語化を引き出すのが上手だから、もやもやしてる大きな課題を相談する
- AIと人間社会への考察
  - 知的労働はもはや人間の専売特許ではないんだなと感じる
  - 少なからず、採用市場にも影響してるだろうし、今後その影響は大きくなるんだろう
  - 10~20年先までAIの進化が進むと、おそらく人間の価値は「人間であること」「肉体があること」「安価な労働力であること」などになってくると思う
  - それがどういう社会を生むのかは正直想像してもしょうがない気がするが、↑を鑑みるとより人間関係にアプローチできる人間性とAIと対話できる専門知識の両立が重要になってくるんだろうなと思う
  - だからエンジニアが絶滅するとは思ってない。形を変えるだけだと思う
  - 形を変える以上、求められるものも変わっていくんだろうというだけ
- AIとエンジニアリングへの考察
  - AIがエンジニアを不要にするとは思わないが、必要なエンジニアの総数は減らすと思う
  - エンジニアに必要になるスキルは、AIを使いこなすこととコードの品質評価ができることになると思う
- まとめ
  - こんなにAIが仕事から日常まで浸透してくるとは思わなかった
  - 意外とAIをまだ全然使ってない人も多いことを知って、びっくりした
