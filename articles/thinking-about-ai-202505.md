AIの進化が目覚ましい昨今、筆者にとってもAIは必要不可欠な存在になっています。これは普段の開発作業におけるサポートのみならず、キャリアの相談、体調の相談、子供の相談など、私生活まで広範な相談をAIにしており、代わりにGoogleで調べ物をする機会が大きく減りました。

日々AIに触れていく過程で、AIの特性やAIに触れる人間側である自分自身の特性について様々な「気付き」を得る機会が増えました。この「気付き」が蓄積し、自分の中で整理しきれなくなってきた感じがしたので、AIについて今思ってることや知っていることを記録を兼ねて記事にすることにしました。

冒頭注意書きした通り、普段Zennに投稿してるより少々フランクで一人語りな記事になると思いますが、ご了承ください。

## 使ってるツール

筆者が2025/05現在、AIツールで使用しているのは以下です。

| ツール名       | 主な用途・使い方                                                                                                                          | 利用頻度・状況   |
| :------------- | :---------------------------------------------------------------------------------------------------------------------------------------- | :--------------- |
| Gemini         | 相談全般                                                                                                                                  | メインで利用     |
| Github Copilot | 参考ファイルがあればAgentモード<br>なければ補完のみ（参考: [AI時代にこそTDDだと思う話](https://zenn.dev/akfm/articles/tdd-with-copilot)） | 実装時           |
| Warp AI        | CLI操作のサポート                                                                                                                         | ターミナルで利用 |
| v0             | プロトタイプ開発や研究目的で利用                                                                                                          | 使う時期のみ     |
| Chat GPT       | Gemini回答に不満な場合のセカンドオピニオン                                                                                                | 稀に利用         |
| Grok           | Xで話題になってる技術の背景や集計など                                                                                                     | 稀に利用         |

Cursorは一時期使ってみたのですが、結局VSCに戻りました。Clineも今は使っておらず、Github CopilotのAgentモードを主に利用しています。

一部用途に応じて使い分けてますが、メインはGeminiです。Chat GPTをメインにしてないのは、コスト・パフォーマンス・Web検索連携の体験やツール連携など鑑みて、非常にコスパがいいと考えてるからです。

## AI全般に対する評価

## Memo

- AI全般に対する評価
  - すでに高い実用性を伴う、非常に強力な「ツール」
    - 総じて非常に賢く、使い手次第で非常に強力なツールになる
    - 弱点としては、「使い手次第」という点
    - エンジニアリングにおいてはDevinやCodexのような自律したAIもいるが、やるべきことや価値観の注入が前提となってるため、やはり「使い手次第」
    - AIと専門的な対話や指示が可能ということは、今後大きなアドバンテージになるだろうと想像してる
    - 逆に、専門的な知識の理解を放棄してAIを使うことはリスクを伴う
  - 倫理的課題、人間社会の需要性
    - ある種自動運転の議論と同じで、「間違ってた時にだれが責任を取るのか」という問題が発生する
    - この責任を取るために、AIの回答を評価し判断できる人間が必要になると思う
  - 自動運転同様、長い時間をかけてこれも変わっていくのかもしれない
- 個人的な使い所とコツ
  - 日々の疑問解消
    - 些細な疑問
    - 子供の食生活や病気
    - ストレスの言語化
  - エンジニアリング
    - 設定ファイルの相談
    - 参考実装がない時: Copilot（Ask）による補完
    - 参考実装がある時: Copilot（Agent）による実装
    - ライブラリ調査や仕様調査
  - キャリアや人生設計の壁打ち
    - 問いや言語化を引き出すのが上手だから、もやもやしてる大きな課題を相談する
- AIの特性
  - 学術的な特性分析ではなく、主観的な定性評価
  - 回答特性
    - Web検索も情報源として使うようになったことで、回答の精度が大きく向上したように感じる
    - 一方で、Webにある情報の組み合わせやクローリングには弱いように感じる
    - 具体的には、Githubにあるからと言って内部実装に応えられるわけではない
    - 誤った情報を鵜呑みにして回答するケースもありうるように思う（ないと言い切れない）
  - 対話特性
    - ツール側の方針なのかもしれないが、より精度の高い答えを導くための対話のラリーが少なく、すぐに答えを出そうとする傾向にある気がする
    - いちいち追加の情報聞かれたらそれはそれでストレスかもしれない
- AIと人間社会への考察
  - 知的労働はもはや人間の専売特許ではないんだなと感じる
  - 少なからず、採用市場にも影響してるだろうし、今後その影響は大きくなるんだろう
  - 10~20年先までAIの進化が進むと、おそらく人間の価値は「人間であること」「肉体があること」「安価な労働力であること」などになってくると思う
  - それがどういう社会を生むのかは正直想像してもしょうがない気がするが、↑を鑑みるとより人間関係にアプローチできる人間性とAIと対話できる専門知識の両立が重要になってくるんだろうなと思う
  - だからエンジニアが絶滅するとは思ってない。形を変えるだけだと思う
  - 形を変える以上、求められるものも変わっていくんだろうというだけ
- AIとエンジニアリングへの考察
  - AIがエンジニアを不要にするとは思わないが、必要なエンジニアの総数は減らすと思う
  - エンジニアに必要になるスキルは、AIを使いこなすこととコードの品質評価ができることになると思う
- まとめ
  - こんなにAIが仕事から日常まで浸透してくるとは思わなかった
  - 意外とAIをまだ全然使ってない人も多いことを知って、びっくりした
